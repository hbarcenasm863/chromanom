Principios de Nomenclatura Inorgánica

Aprende los fundamentos: iones, estados de oxidación, tipos de compuestos y los tres sistemas de nomenclatura

El sistema de colores 2+matiz

La nomenclatura inorgánica describe compuestos formados por iones. En el modelo 2+matiz cada tipo de ion recibe un color fijo, lo que hace visible la estructura del compuesto de un vistazo.

① Ámbar — Catión (metal) Átomo que pierde electrones y queda con carga positiva. Va primero en fórmula y nombre. Ejemplos: Na⁺, Ca²⁺, Fe³⁺, Cu²⁺, Al³⁺
② Azul — Anión simple No metal que gana electrones. En el nombre recibe la terminación -uro. Forma sales binarias. Ejemplos: Cl⁻ (cloruro), S²⁻ (sulfuro), O²⁻ (óxido)
③ Violeta — Grupo OH⁻ El grupo hidroxilo es un anión poliatómico especial. Siempre se llama hidróxido. Forma bases. Ejemplos: NaOH → hidróxido de sodio, Ca(OH)₂ → hidróxido de calcio
④ Teal — Oxoanión Anión que contiene oxígeno. Tiene nombre propio que hay que memorizar. Forma sales oxigenadas. Ejemplos: SO₄²⁻ (sulfato), NO₃⁻ (nitrato), PO₄³⁻ (fosfato)

Compuestos iónicos: lectura de fórmulas

Un compuesto iónico está formado por un catión y un anión. Los subíndices se ajustan para que las cargas se cancelen:

NaCl = Na⁺ (×1) + Cl⁻ (×1) · cargas: +1 – 1 = 0 ✓
CaCl₂ = Ca²⁺ (×1) + Cl⁻ (×2) · cargas: +2 – 2 = 0 ✓
Al₂O₃ = Al³⁺ (×2) + O²⁻ (×3) · cargas: +6 – 6 = 0 ✓
Fe₂(SO₄)₃ = Fe³⁺ (×2) + SO₄²⁻ (×3) · cargas: +6 – 6 = 0 ✓

Valencias fijas y variables

Los metales del bloque s (Na, K, Ca, Mg, Al) tienen valencia fija — siempre la misma. Los metales de transición (Fe, Cu, Pb, Mn, Cr) tienen valencia variable y necesitan un sistema que indique cuál se usa.

Estados de oxidación (EO)

El estado de oxidación es el número de electrones que un átomo ha perdido, ganado o compartido. Es fundamental para nombrar compuestos de metales con valencia variable.

Reglas de asignación

1
Elemento puro:

EO = 0. Ej: Fe(s), O₂(g), Na(s) → todos tienen EO = 0

2
Ión monoatómico:

EO = carga del ión. Ej: Na⁺ → EO = +1 · Fe³⁺ → EO = +3

3
Oxígeno:

EO = −2 en casi todos los compuestos. Excepción: en O₂ es 0; en peróxidos (H₂O₂) es −1

4
Hidrógeno:

EO = +1 cuando va con no metales. EO = −1 en hidruros metálicos (NaH, CaH₂)

5
Regla de la suma:

En molécula neutra: suma de todos los EO = 0. En ión poliátomico: suma = carga del ión

Ejemplos de cálculo

Fe₂O₃ — ¿cuál es el EO del hierro?
Fe: x  |  O: −2
2(x) + 3(−2) = 0
2x − 6 = 0  →  x = +3
∴ El Fe tiene EO = +3 → es Fe³⁺ (hierro III)
Cu₂O — ¿cuál es el EO del cobre?
Cu: x  |  O: −2
2(x) + 1(−2) = 0
2x = +2  →  x = +1
∴ El Cu tiene EO = +1 → es Cu⁺ (cobre I)
KMnO₄ — ¿cuál es el EO del manganeso?
K: +1  |  Mn: x  |  O: −2
+1 + x + 4(−2) = 0
x = 8 − 1  →  x = +7
∴ El Mn tiene EO = +7 → permanganato
H₂SO₄ — ¿cuál es el EO del azufre?
H: +1  |  S: x  |  O: −2
2(+1) + x + 4(−2) = 0
x = 8 − 2  →  x = +6
∴ El S tiene EO = +6 en el ácido sulfúrico

Nomenclatura Sistemática / IUPAC

Usa prefijos numéricos para indicar la cantidad de átomos de cada elemento. Es la más moderna, no necesita conocer valencias.

Reglas generales

  1. Se nombra primero el elemento menos electronegativo (el metal o el elemento de la izquierda)
  2. Se usa prefijo para indicar la cantidad de átomos de cada elemento
  3. Si el prefijo termina en vocal y el nombre empieza en vocal, se omite la vocal del prefijo: monooxido → monóxido
  4. El segundo elemento termina en -uro (aniones simples) o -ido (óxidos, anhídridos)
CO → monóxido de carbono (1 O, 1 C)
CO₂ → dióxido de carbono (2 O, 1 C)
SO₃ → trióxido de azufre
P₂O₅ → pentaóxido de difósforo
Cl₂O₇ → heptaóxido de dicloro
N₂O₃ → trióxido de dinitrógeno
FeCl₃ → tricloruro de hierro (para sales binarias)

Nomenclatura Stock

Indica el estado de oxidación del metal con números romanos entre paréntesis, justo después del nombre del metal. Es el estándar internacional actual.

FeCl₂ → cloruro de hierro (II)  [Fe tiene EO = +2]
FeCl₃ → cloruro de hierro (III)  [Fe tiene EO = +3]
Cu₂O → óxido de cobre (I)  [Cu tiene EO = +1]
CuO → óxido de cobre (II)  [Cu tiene EO = +2]
PbO₂ → óxido de plomo (IV)  [Pb tiene EO = +4]

Para metales de valencia fija (Na, Ca, Al…) no se usa número romano porque no hay ambigüedad. El número romano solo aparece cuando el metal puede tener más de una valencia.

Nomenclatura Tradicional (Clásica)

Usa sufijos para indicar si el metal está en su valencia menor o mayor. Es más antigua y sigue usándose en algunos contextos hispanohablantes.

Cuando es la valencia…SufijoEjemplo con Fe
Menor (la más baja)-osoferroso (+2) → cloruro ferroso
Mayor (la más alta)-icoférrico (+3) → cloruro férrico
FeCl₂ → cloruro ferroso (Fe²⁺ = valencia menor de Fe)
FeCl₃ → cloruro férrico (Fe³⁺ = valencia mayor de Fe)
Cu₂O → óxido cuproso (Cu⁺ = valencia menor de Cu)
CuO → óxido cúprico (Cu²⁺ = valencia mayor de Cu)
SnCl₂ → cloruro estannoso (Sn²⁺)
SnCl₄ → cloruro estánnico (Sn⁴⁺)

Nota: La nomenclatura Stock es el estándar internacional recomendado por IUPAC para compuestos de metales con valencia variable. La Tradicional se mantiene en programas de bachillerato hispanoamericano.

¿Por qué SO₃ se llama "anhídrido" en Tradicional pero "óxido de azufre (VI)" en Stock?
La nomenclatura Tradicional distingue entre óxidos (metal + O) y anhídridos (no metal + O). Stock e IUPAC unifican ambos bajo la palabra "óxido", usando el número de oxidación o los prefijos para diferenciarlos. En el aula se usan los dos sistemas; es importante reconocer ambas formas.
